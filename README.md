# LanguageLearningTracker

A simple Javascript language learning tracker that stores the user's hours and minutes for different categories per day. It also displays stats from anki if the user has anki and the Anki-Connect plugin, but this is optional.

## Installation

1. Clone the repository
    'git clone https://github.com/emilykrohn/LanguageLearningTracker.git'
2. Open the project directory in VSCode
3. Use the Live Server extention to open the website

## Access through browser
https://emilykrohn.github.io/LanguageLearningTracker/

## Usage

    - Click on a date on the calendar to open a form
    - Add the number of hours and minutes to each field
    - View stats from anki if it's connected to the website

## Technologies Used

    - HTML5
    - CSS
    - JavaScript (Vanilla)

## Anki-Connect
Anki-Connect is used to display stats from the user. Anki needs to be installed first and then the Anki-Connect plugin can be added. When the website is running, anki needs to be opened for it to work.
    -https://git.sr.ht/~foosoft/anki-connect
